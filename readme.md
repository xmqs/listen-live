# listen-live
### 主要功能：听
Terminalm 控制台启动 douyu/huya 直播

集成ffplay播放器，播放直播流的音频
# 开发原因
### 听
主要自己想上班听枫哥直播（rua）

大部分程序员上班能听，但是不太方便看直播（狗头）

满足听直播需求

# 直播流来源
https://github.com/YanG-1989/m3u
# 食用方法

npm i listen-live

listen-live

选择平台 比如：douyu

输入room 房间号：9999
